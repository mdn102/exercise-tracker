### Setup DB
```
# Add .env file in root folder  
ATLAS_URI="fill in your url of mongodb at here"
SKIP_PREFLIGHT_CHECK=true
```

### Usage

```
 # root folder
npm, install

# cd backend
 npm install
 cd ..
 
  
 # Build client
 npm run build
 
 # run 
 npm start
```
